1. Objet

Cette application (« Claude Desktop MCP ») permet la connexion sécurisée entre Claude Desktop (Anthropic) et QuickBooks Online (Intuit) via l'API Intuit OAuth 2.0 pour la gestion des données comptables.

2. Usage

Cette application est développée pour un usage interne exclusif par La Réserve Naturelle. Elle n'est pas distribuée au public et n'est pas disponible sur l'App Store Intuit.

3. Données

L'application accède aux données comptables QuickBooks Online (comptes, factures, dépenses, rapports) uniquement dans le cadre des opérations internes de La Réserve Naturelle. Toutes les données sont traitées localement sur l'appareil de l'utilisateur.
