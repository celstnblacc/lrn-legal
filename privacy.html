Politique de confidentialité

Application : Claude Desktop MCP
Éditeur : La Réserve Naturelle
Dernière mise à jour : 12 février 2026

1. Introduction

Cette politique de confidentialité décrit comment l'application Claude Desktop MCP traite les données lorsqu'elle est connectée à QuickBooks Online via l'API Intuit.

2. Données collectées

L'application accède aux données comptables QuickBooks Online suivantes dans le cadre de son fonctionnement :

  • Comptes et plan comptable
  • Factures et devis
  • Dépenses et achats
  • Clients et fournisseurs
  • Rapports financiers

3. Traitement des données

Toutes les données sont traitées localement sur l'appareil de l'utilisateur. L'application utilise le protocole MCP (Model Context Protocol) qui communique via stdin/stdout — aucune donnée n'est transmise sur le réseau au-delà de la connexion sécurisée à l'API Intuit.

4. Stockage

Les identifiants d'authentification (Client ID, Client Secret, Refresh Token) sont stockés dans un fichier .env local sur l'appareil de l'utilisateur. Aucune base de données externe n'est utilisée. Aucune donnée n'est stockée dans le cloud par cette application.

5. Partage avec des tiers

Aucune donnée n'est partagée avec des tiers. L'application communique uniquement avec l'API QuickBooks Online d'Intuit via une connexion OAuth 2.0 sécurisée.

6. Sécurité

L'application met en œuvre les mesures de sécurité suivantes :

  • Authentification OAuth 2.0 avec tokens à durée limitée
  • Stockage local exclusif des identifiants
  • Communication locale via le protocole MCP
  • Code source ouvert et auditable (licence MIT)

7. Vos droits

Étant donné que cette application est utilisée exclusivement en interne par La Réserve Naturelle, les données traitées sont sous le contrôle direct de l'entreprise. Pour toute question ou demande de suppression de données, veuillez nous contacter.
